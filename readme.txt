1. Padding

	•	Defines the space between the content of an element and its border.
	•	Example: padding: 10px; adds 10 pixels of space inside the element’s border.

2. Margin

	•	Defines the space between an element and its surroundings.
	•	Example: margin: 10px; adds 10 pixels of space outside the element’s border.

3. Font-Size

	•	Determines the size of the text in an element.
	•	Example: font-size: 16px; sets the text size to 16 pixels.

4. Font-Weight

	•	Controls the thickness or boldness of the text.
	•	Values: normal, bold, lighter, bolder, or numerical values like 100, 400, 700.
	•	Example: font-weight: bold; makes the text bold.

5. Color

	•	Specifies the color of the text.
	•	Example: color: red; changes the text color to red.

6. Background-Color (bg-color)

	•	Sets the background color of an element.
	•	Example: background-color: blue; makes the element’s background blue.

7. Box-Shadow

	•	Adds shadow around an element.
	•	Syntax: box-shadow: offset-x offset-y blur-radius spread-radius color;
	•	Example: box-shadow: 2px 2px 5px grey; creates a grey shadow with a slight offset and blur.

8. Gap

	•	Used in grid or flexbox layouts to define the spacing between rows and columns.
	•	Example: gap: 10px; creates a 10-pixel gap between items in a grid or flexbox.

9. Display Flex

	•	Sets the display property to flex, enabling a flex container for layout.
	•	Example: display: flex; makes the element a flex container.

10. Align Items Center

	•	Aligns flex items vertically in the center of the container.
	•	Example: align-items: center;.

11. Justify Content Center

	•	Aligns flex items horizontally in the center of the container.
	•	Example: justify-content: center;.

12. Justify Content Between

	•	Distributes flex items with equal space between them, with no space at the ends.
	•	Example: justify-content: space-between;.

13. Border Radius

	•	Rounds the corners of an element.
	•	Example: border-radius: 10px; creates rounded corners with a 10-pixel radius.
